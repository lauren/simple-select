  • Use simpleSelect("div") to give all divs a border.
  • Use simpleSelect("div.container a") to bold links that are children of div.container.
  • Use simpleSelect("div.square.first-half") to give a dashed border to divs that have the class square AND the class first-half.
  • Use simpleSelect("div.container") to make the border of divs with the class .container 2px wide.
  • Use simpleSelect("div#one") to give the div with the ID #one a red background.
  • Use simpleSelect("div#two a.normal") to give the an underline to the link with class .normal inside the div with ID #two.
  • Use simpleSelect(".first-half .normal") to italicize any text inside an element with the class .normal that has a parent class .first-half.
  • Use simpleSelect("#five .normal") give a yellow background to any element with the ID #link-five inside an element with the ID #five.
  • Use simpleSelect(".container .inner-container .loner-link") give a green background to any element with the ID #loner inside an element with the class .inner-container inside an element with the class .container.
  • Use simpleSelect("div.square a.special") to find the links with the class .special inside divs with the class .square and make their links purple.
  • Use simpleSelect(".square") increase the line height in all elements with the class .square.
  • Use simpleSelect(".square.first-half") to make the text bold in any div with the class .first-half AND the class .square.
  • Use simpleSelect("#two") to give any element with the id #two a yellow background.
